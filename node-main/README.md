# One-Click Installation (Recommended)

The easiest way to install PasarGuard Node is using the automated installation script:

```bash
sudo bash -c "$(curl -sL https://github.com/ASTRACAT2022/ParsadGuard-for-singbox/raw/main/node-main/install.sh)" @ install
```
